OptimoRoute Sorter Application
Version: 1.0.0
Created: 2024

This package contains everything needed to run the OptimoRoute Sorter application.

Features:
- OptimoRoute API integration for fetching scheduled deliveries
- PDF processing and driver-based sorting
- OCR support for image-based PDFs
- Professional GUI interface
- Automated delivery sequence processing
